Es necesario cumplir con los siguientes requisitos para la creación del CRUD:

Backend

  • Debe ser construido en c# .net 5.
  • Las funciones crud (insertar, eliminar, modificar, seleccionar) deben hacerse a través de una apirest.
  • La base de datos a usar debe ser sql server.
  • Usar ORM como EntityFramework o Nhibernate para el acceso a datos.
  • Usar mínimo un patrón de diseño.
  • Realizar el montaje de la solución en un contenedor Docker.

Frontend

  • Realizar el consumo de los servicios del lado cliente usando alguna librería o framework como Angular, Svelte, Vue, React, Jquery o el de su preferencia.

Utilities

  • Compartí el desarrollo de la actividad a través de un repositorio de código fuente tales como GitHub, GitLab, Bitbucket o el de su preferencia.
  • Realizar el montaje de las soluciones con Docker (Opcional).
